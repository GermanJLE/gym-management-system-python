# 🏋️ Gymrats – Gym Management System

## 📖 Overview

Gymrats is a console-based gym management system developed in Python using Object-Oriented Programming (OOP).

The application allows users to manage gym members, instructors, classes, memberships and reports through an interactive menu.

## 🚀 Features

- Member management
- Instructor management
- Class management
- Membership management
- Reports
- Console-based interface

## 🛠️ Technologies

- Python
- Object-Oriented Programming (OOP)
- CSV Files

## 📚 What I Learned

This project helped me strengthen my understanding of Object-Oriented Programming by applying concepts such as encapsulation, inheritance, composition, aggregation, modular programming and data validation.
